# 📊 PathTracker 业务路径分析引擎

**PathTracker** 是一款专为业务运营和数据分析团队打造的“轻量级、高私密性”数据可视化中台。无需复杂的后端环境配置，只需一个浏览器，即可将繁杂的 Excel/CSV 业务数据转化为直观的流转路径图。

## 🛡️ 核心设计理念：绝对的数据安全
市面上大多数数据分析工具都需要将企业核心业务数据上传至第三方云端，存在极大的数据泄露风险。
PathTracker 采用 **纯前端（Browser-Native）计算架构**，所有的数据脱水、交叉比对和图表渲染，**100% 在用户本地电脑的浏览器内存中完成**，彻底断绝网络传输，满足最高级别的商业机密保护要求。

## ✨ 核心特性
* **🧩 乐高式维度搭建：** 告别写 SQL 代码，全图形化界面。拖拽业务字段（如：分配渠道 -> 第一节课到课 -> 最终转化），秒出分析图表。
* **📈 全景可视化视图：** 内置三大主流分析模型，一键自由切换：
    * **桑基流转图 (Sankey)：** 洞察全局流量的走向与分流比例。
    * **树状发散图 (Tree)：** 追溯具体节点的转化深度与转化率。
    * **漏斗转化图 (Funnel)：** 宏观掌控各层级的流失情况。
* **⚖️ 动态多轨对比模式：** 独创的无限画布设计，支持同时拉起 A/B/C/D 多组数据看板进行独立筛选与横向对比，精准定位变量差异。
* **⚙️ 强悍的底层抗压：** 搭载防卡死熔断机制与超大维度自动折叠技术，几万条业务流水瞬间解析，拒绝假死。

## 🎯 典型适用场景
* **用户多维度转化分析：** 不同维度，无限交叉，查看数据差异。
* **销售 SOP 漏斗复盘：** 对比不同销售小组、不同渠道来源的用户转化效率差异。
* **用户增长路径洞察：** 快速定位业务流程中的“断点”与“流失重灾区”。

## 🚀 快速上手
本项目为零依赖的单页面应用（SPA）。
1. 克隆或下载本项目代码。
2. 双击打开 `index.html`，或将其托管至任意静态服务器（如 Netlify, Vercel, GitHub Pages）。
3. 将包含表头的 `.csv` 或 `.xlsx` 文件拖入工作台。
4. 拼接业务维度，开始你的数据探索之旅！

## 🛠️ 技术栈
* **UI 框架:** HTML5 + Tailwind CSS
* **可视化引擎:** Apache ECharts 5.5
* **数据脱水层:** PapaParse (CSV) + SheetJS (Excel)
